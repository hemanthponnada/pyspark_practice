# 🚀 Mastering PySpark

A structured, hands-on journey to mastering **PySpark** for Data Engineering, Data Science, and Distributed Machine Learning.

This repository is a growing collection of notebooks and examples designed to build strong fundamentals and advanced expertise in Spark.

---

## 🎯 Objective

The goal of this repository is to:

- Develop deep conceptual clarity in Spark
- Master distributed data processing
- Build production-ready transformations
- Strengthen feature engineering for ML
- Understand performance and optimization

This is not just syntax practice — it focuses on thinking in distributed systems.

---

## 📚 What This Repository Will Cover

As this repository grows, it will include:

### 🔹 Core Spark Fundamentals
- Spark architecture overview
- RDD vs DataFrame vs Dataset
- Lazy evaluation
- Transformations vs Actions
- Spark execution plans

### 🔹 DataFrame Transformations
- Filtering & aggregations
- Joins (inner, left, broadcast, skew handling)
- Complex groupBy operations
- Column expressions & UDFs

### 🔹 Window Functions
- Partitioning & ordering
- Ranking functions
- Lag/Lead
- Rolling aggregations
- Time-series feature engineering

### 🔹 Advanced Topics
- Performance tuning
- Shuffle optimization
- Partitioning strategies
- Caching vs checkpointing
- Delta Lake basics
- Structured Streaming (future)

### 🔹 ML & Forecasting Features
- Lag features
- Rolling statistics
- Volatility & momentum
- Leakage-safe feature engineering
